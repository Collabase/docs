<Update label="v0.8.3" description="Latest Release">
  ## What's new in v0.8.3?

  In this release, we focused on improving the user interface and adding new capabilities for project tracking and documentation.

  ### ✨ New features & improvements
  * **Kanban board updates:** Added new Kanban features including colored filters, swimlanes, customizable card fields, and quick actions.
  * **Document editor:** You can now embed a *Collabase Tasks* macro directly inside your wiki pages.
  * **Registry:** Linked tasks are now clearly visible in the detail view of any custom Registry object.

  ### 🎨 Layout & UX
  * **Navigation:** The app sidebar and header layout were redesigned for a cleaner, more intuitive user experience.
  * **Dashboards:** The Home and Space dashboards have been reorganized to help you find your work faster.

  ### 🛠 Stability & Maintenance
  * **Automation:** Stability improvements for workflow executions.
  * **System updates:** General performance updates across the web interface, backend, and AI services.
  * **Bug fixes:** Resolved various issues in the demo environment.

  ### 🔒 Security & Vulnerabilities

  * Performed routine security updates across underlying backend services (FastAPI) and third-party AI components (LangChain).
  * Maintained frontend and core UI dependencies to ensure ongoing platform stability and security.
</Update>
