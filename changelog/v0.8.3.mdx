<Update label="v0.8.3" description="Latest Release">
  ## What's new in v0.8.3?

  In this release, we focused on improving the user interface and adding useful new features for projects and documents.

  ### ✨ New Features & Improvements
  * **Kanban Board Updates:** We've added extensive Kanban features, including colored filters, swimlanes, customizable card fields, and new quick actions. (#251)
  * **Document Editor:** There is now a brand-new *Collabase Tasks* macro directly available in the Document Editor. (#246)
  * **Registry:** Linked tasks are now clearly displayed in the detail view of Registry objects. (#243)

  ### 🎨 Refactoring & UX
  * **Layout & Navigation:** The app sidebar and basic header layout were completely restructured to provide an even better user experience. (#242)
  * **Cockpits:** The Home Cockpit and Space Cockpit have been completely revamped and cleaned up. (#252)
  * **Automation Engine:** Internal refactoring to ensure a more stable automation experience. (#239)

  ### 🐛 Bug Fixes
  * Various bug fixes in the demo environment. (#224)

  ### 📦 Maintenance & Dependency Updates
  * **Frontend & Core:** Various security and feature updates for Vite, React Flow (`@xyflow/react`), Radix-UI, JSDom, and Playwright.
  * **Backend & AI Engine:** Updates for FastAPI, Pytest, and various LangChain modules (Groq & OpenAI) in the Collabase microservices.
</Update>
