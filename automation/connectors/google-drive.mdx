# Google Drive

Manage files and folders in Google Drive from your automations — create folders, upload files from URLs, share files with external users, and move items between folders.

## Authentication

**Auth type:** OAuth 2.0

Enable the **Google Drive API** in [Google Cloud Console](https://console.cloud.google.com/). Generate an OAuth 2.0 access token with `https://www.googleapis.com/auth/drive` scope.

| Credential | Description |
|---|---|
| **Access Token** | OAuth 2.0 access token with Drive scope |

## Triggers

| Trigger | When it fires |
|---|---|
| **File Created** | A new file is added to a monitored folder |
| **File Updated** | An existing file is modified |
| **File Deleted** | A file is removed |
| **File Shared** | A file is shared with a new user |
