# Google Calendar

Manage events across Google Calendar — from creating meetings to checking availability before scheduling. Works with any Google Calendar account (personal or Google Workspace).

## Authentication

**Auth type:** OAuth 2.0

Create OAuth 2.0 credentials in [Google Cloud Console](https://console.cloud.google.com/) → APIs & Services → Credentials. Enable the **Google Calendar API**. Generate a token using the OAuth 2.0 flow with `https://www.googleapis.com/auth/calendar` scope.

| Credential | Description |
|---|---|
| **Access Token** | OAuth 2.0 access token (refresh via your OAuth flow) |

## Triggers

| Trigger | When it fires |
|---|---|
| **Event Created** | A new event is added to a calendar |
| **Event Updated** | An existing event is modified |
| **Event Deleted** | An event is removed from the calendar |
| **Event Starting Soon** | An event begins within a configured window |
