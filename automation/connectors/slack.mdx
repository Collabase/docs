# Slack

Send messages to Slack channels and direct message users from any automation. Slack is one of the most common connectors — use it to surface alerts, approval links, and workflow updates directly in your team's conversations.

## Authentication

**Auth type:** Bot token (OAuth 2.0 installation)

Create a Slack app at [api.slack.com/apps](https://api.slack.com/apps). Add the following bot token scopes: `chat:write`, `im:write`, `channels:join`, `reactions:write`, `files:write`. Install the app to your workspace and copy the **Bot User OAuth Token** (`xoxb-...`).

| Credential | Description |
|---|---|
| **Bot Token** | Your Slack bot OAuth token, starting with `xoxb-` |

## Triggers

| Trigger | When it fires |
|---|---|
| **Message Posted** | A new message is posted in a monitored channel |
| **Direct Message Received** | A user sends a DM to the bot |
| **Reaction Added** | A reaction emoji is added to any message |
| **App Mention** | A user mentions the bot with `@botname` |

<Note>
  Triggers require you to configure a Slack webhook URL in your app's **Event Subscriptions** settings, pointing to your Collabase instance's webhook endpoint.
</Note>

## Actions

    | Action | Key inputs | Key outputs |

    | **Send Message** | `channel`, `text`, `username`, `iconEmoji`, `blocks` | ok, ts, channel |

    | **Update Message** | `channel`, `ts`, `text` | ok, ts |
    | **Delete Message** | `channel`, `ts` | ok |
  </Accordion>
  <Accordion title="Reactions & files">
    | Action | Key inputs | Key outputs |
    |---|---|---|
    | **Add Reaction** | `channel`, `ts`, `emoji` | ok |
    | **Upload File** | `channels`, `fileUrl`, `filename`, `title`, `initialComment` | fileId, permalink |
  </Accordion>
  <Accordion title="Channel management">
    | Action | Key inputs | Key outputs |
    |---|---|---|
    | **Create Channel** | `name`, `isPrivate` | id, name |
    | **Invite User to Channel** | `channel`, `userId` | ok |
    | **Get Channel Info** | `channel` | id, name, topic, memberCount |
    | **List Members** | `channel` | members[], count |
  </Accordion>
</AccordionGroup>

<Tip>
  Use the channel's **ID** (e.g. `C012AB3CD`), not its display name, for reliable delivery. Find the ID by right-clicking the channel in Slack and selecting **Copy link** — the last segment of the URL is the ID.
</Tip>
