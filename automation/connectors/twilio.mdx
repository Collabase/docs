# Twilio

Send SMS and MMS messages to any phone number worldwide, initiate WhatsApp messages through Twilio's sandbox or approved numbers, and trigger voice calls with TwiML scripts.

## Authentication

**Auth type:** Basic (Account SID + Auth Token)

Find your credentials in the [Twilio Console](https://console.twilio.com/) dashboard.

| Credential | Description |
|---|---|
| **Account SID** | Your Twilio account SID (starts with `AC`) |
| **Auth Token** | Your Twilio auth token |
| **From Number** | The Twilio phone number or WhatsApp sender to use |

## Actions

| Action | Key inputs | Key outputs |
|---|---|---|
| **Send SMS** | `to`, `body` | sid, status, to |
| **Send MMS** | `to`, `body`, `mediaUrl` | sid, status, to |
| **Send WhatsApp** | `to`, `body` (prefix `whatsapp:` on both from/to is handled automatically) | sid, status |
| **Make Voice Call** | `to`, `twimlUrl` or `twiml` | sid, status, direction |
| **Lookup Number** | `phoneNumber` | phoneNumber, countryCode, carrier`, lineType |

<Warning>
  Voice calls use TwiML to define what happens when the call connects. Either host a TwiML XML file and pass its URL, or provide the TwiML string directly in the `twiml` field. See the [Twilio TwiML docs](https://www.twilio.com/docs/voice/twiml) for the full syntax.
</Warning>
