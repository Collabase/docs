# HubSpot

Sync CRM data, automate pipeline stages, and trigger workflows from HubSpot events. Connect your Collabase automations to HubSpot's full contact lifecycle.

## Authentication

**Auth type:** API Key (Private App Token)

Create a private app in [HubSpot Settings → Integrations → Private Apps](https://app.hubspot.com/private-apps). Grant it the scopes you need: `crm.objects.contacts`, `crm.objects.deals`, `crm.objects.companies`, etc.

| Credential | Description |
|---|---|
| **Access Token** | Private app token from HubSpot (starts with `pat-`) |

## Triggers

| Trigger | When it fires |
|---|---|
| **Contact Created** | A new HubSpot contact is created |
| **Deal Created** | A new deal is added to the pipeline |
| **Deal Stage Changed** | A deal moves to a different pipeline stage |
| **Deal Won** | A deal is marked as closed-won |
| **Deal Lost** | A deal is marked as closed-lost |
| **Company Created** | A new company record is created |
| **Ticket Created** | A new support ticket is opened |
| **Contact Property Changed** | A contact's property value is updated |
