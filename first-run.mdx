## Logging in for the first time

Once the installation is complete, open your Collabase URL in a browser. Log in with the admin email and password you set during installation.

<Steps>
  <Step title="Log in">
    Go to your Collabase URL and enter your admin email and password.
  </Step>
  <Step title="Set a new password">
    You will be asked to choose a new password on your first login. Enter and confirm it, then log in again with the new password.
  </Step>
  <Step title="Create your first Space">
    After logging in, click **New Space** to create your first workspace. Give it a name — this is the container for your team's pages, tasks, and test cases.
  </Step>
  <Step title="Invite your team">
    Go to **Admin → Users** and add your colleagues. Each user gets access to the Spaces you assign them to.
  </Step>
</Steps>

---

## Demo data

If you chose to install demo data during setup, you will find pre-built Spaces and example content already in your instance. This lets you explore the product before adding real content.

To remove demo data, delete the Spaces from the Collabase admin panel or from within the app.

## Next steps

| What to do | Where |
|---|---|
| Invite users | Admin → Users |
| Set up email (SMTP) | Admin → Settings → Mail |
| Connect your identity provider (SSO, LDAP) | Admin → Identity Providers |
| Configure AI features | Admin → Settings → AI |
| Enable or disable features per Space | Space → Settings → Apps |
| Change the instance name or logo | Admin → Settings → Branding |
