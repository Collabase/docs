# Quick Links

Quick Links are pinned shortcuts displayed on the Intranet home. They give the whole team one-click access to frequently used tools, documents, and external services without searching.

## Types of quick links

| Type | Visibility |
|---|---|
| **Organization links** | Set by admins — visible to everyone |
| **Personal links** | Set by individual users — visible only to them |

## Adding organization quick links (Admin)

1. **Admin → Intranet Config → Quick Links → + Add Link**
2. Enter a title, URL, and optionally an icon
3. Drag to reorder
4. Save

These appear for all users in the Quick Links widget on the Intranet home.

## Adding personal quick links

1. **Intranet → Quick Links → + Add Personal Link**
2. Enter a title and URL
3. Save

Personal links appear in your own Quick Links panel below the organization links.

## Link icons

Quick Links support favicon auto-fetch — paste the URL and Collabase fetches the site favicon automatically. You can also upload a custom 32×32 icon.
