# Events

The Intranet Events section surfaces upcoming company-wide events to everyone in the organization — team meetings, company all-hands, workshops, deadlines, and social events.

## Viewing events

The Events widget on the Intranet home feed shows upcoming events for the next 30 days. Click any event to see the full details and any attached notes.

The **Events** tab shows a full calendar view and a list of all upcoming events.

## Creating an event

Events can be created by Admins or users with the **Event Manager** role:

1. **Intranet → Events → + New Event**
2. Fill in:
   - **Title** — required
   - **Date & Time** — required
   - **End Time** — optional
   - **Location** — physical address or video call link
   - **Description** — rich text with full formatting support
   - **All day** — toggle for full-day events
3. Click **Create**

The event immediately appears in the Intranet feed for all users.

## Event reminders

Users can click **Add to my calendar** on any event to download an `.ics` file compatible with Google Calendar, Outlook, and Apple Calendar.

## Admin management

Admins can edit or delete any event under **Admin → Intranet → Events**. Individual users can only delete events they created.
