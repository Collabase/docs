Collabase is a **self-hosted collaboration OS** that keeps your data on your own servers. It combines a rich-text knowledge base, task management, quality assurance, visual automation, local AI, and an internal communication hub — all in one platform. A shared permission model and unified search connect everything.

## Get started in minutes

<Steps>
  <Step title="Install Collabase">
    Run the installer on any Linux server (or macOS / Windows WSL2) with Docker installed:
    ```bash
    curl -fsSL https://raw.githubusercontent.com/Collabase/collabase/main/deployment/install.sh | bash
    ```
    The installer is interactive — it will ask for your domain, admin email, and password, then generate all secrets and start all containers automatically.
  </Step>
  <Step title="Log in and set your password">
    Navigate to your Collabase URL and log in with the admin email and password you set during installation. You will be prompted to choose a new password on first login.
  </Step>
  <Step title="Create your first Space">
    Spaces are the top-level containers for your team's work. Create one for a project, department, or product area.
  </Step>
  <Step title="Invite your team">
    Go to **Admin → Users** and add team members. Assign them to Spaces with the appropriate role.
  </Step>
</Steps>
