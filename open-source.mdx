# Open Source & BSL 1.1

Collabase source code is publicly available at [github.com/Collabase/collabase](https://github.com/Collabase/collabase).

## The license

Collabase is licensed under the **Business Source License 1.1 (BSL 1.1)** with an automatic conversion to **Apache 2.0 on 1 April 2031**.

The BSL 1.1 allows you to:

- ✅ View, fork, and study the source code
- ✅ Run Collabase for internal business use — any number of users
- ✅ Modify Collabase for your own internal deployment
- ✅ Contribute back via pull requests

The BSL 1.1 does **not** allow:

- ❌ Offering Collabase as a hosted service to third parties (SaaS)
- ❌ Reselling Collabase or a fork commercially

After 1 April 2031, all restrictions lift and every past version becomes Apache 2.0.

## Why BSL 1.1?

1. **Sustainable development** — enterprise licenses fund core development
2. **Full transparency** — you can read every line of code you run
3. **No cloud lock-in** — we retain the right to build a hosted offering
4. **Long-term open path** — Apache 2.0 in 2031 is a firm commitment
