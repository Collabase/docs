# Collabase Docs

<img src="/images/docs.png" alt="Collabase Docs interface" style={{ borderRadius: '8px', marginBottom: '24px' }} />

Collabase Docs is the built-in knowledge base for every Space. It's where teams write, review, and maintain all their structured knowledge — technical specs, runbooks, design decisions, meeting notes, onboarding guides, and more.

Every document is a **page** — rich text with a status, author, version history, permissions, and a place in the Space's nested tree.

## The editor

The Collabase editor supports:

| Feature | How |
|---|---|
| **Rich text** | Bold, italic, underline, strikethrough, inline code |
| **Headings** | H1–H3 with auto-generated anchor links |
| **Code blocks** | Syntax highlighted, with language picker |
| **Tables** | Full table editing with row/column add, delete, and resize |
| **Images** | Upload or paste |
| **Callouts** | Info, warning, tip, danger callout blocks |
| **Embeds** | YouTube, Figma, Loom, and any URL-based embed |
| **@Mentions** | Mention team members — they get an in-app notification |
| **Slash commands** | Type `/` to insert any block type |
| **Markdown shortcuts** | `**bold**`, `# Heading`, `` `code` `` |

## Page status flow

```
DRAFT → IN REVIEW → PUBLISHED → ARCHIVED
```

- **DRAFT** — visible to editors only
- **IN REVIEW** — flagged for review; visible to members
- **PUBLISHED** — the live, canonical version
- **ARCHIVED** — retained for history but not shown in navigation

## AI features

The **AI Assistant** in Docs provides:
- **Slash → AI: Improve** — rewrite selected text to be clearer
- **Slash → AI: Translate** — translate selected text to another language
- **Slash → AI: Summarize** — generate a summary of the whole page
- **Slash → AI: Continue** — generate continuation based on context

AI runs via your configured AI model (Ollama or OpenAI) — see **Admin → AI Configuration**.

## Integration with other Collabase apps

- **Test Management** — Link test specs in Docs to test cases
- **Projects** — Attach pages as project documentation
- **Automation** — Trigger automations when pages are created or updated
- **AI Brain** — Published pages are indexed and searchable via the AI Brain
