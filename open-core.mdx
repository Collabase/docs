# Open Core Model

Collabase is built on an open-core model: the platform foundation is publicly available under the Business Source License 1.1, while advanced enterprise features are distributed as a separately licensed module.

## Activating enterprise

<Steps>
  <Step title="Purchase">Purchase a license at [collabase.com/enterprise](https://collabase.com/enterprise)</Step>
  <Step title="Enter key">Enter your license key in **Admin → License**</Step>
  <Step title="Install">Click **Install Enterprise Module** — Collabase installs `@collabase/enterprise` automatically</Step>
  <Step title="Restart">Restart your server — enterprise features are immediately active</Step>
</Steps>

The enterprise module is distributed via a private npm registry. Your license key contains a scoped access token used transparently during installation.

## Grace period

If your license expires, Collabase enters a 30-day grace period. Enterprise features remain active and a renewal reminder is shown in the admin panel. After the grace period, the platform switches back to Community Edition automatically.
